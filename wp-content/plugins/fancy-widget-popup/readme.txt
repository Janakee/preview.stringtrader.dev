== Description ==

This plugin will give you option to add any widget in sidebar provided by the plugin. It will PopUp when user reaches at footer of the page of website. This will help out user to gain attention after watching page completely.

For Demo check page
http://www.webkhazana.com/plugin/fancy-widget-popup/

== Installation ==

1. Upload fancy-widget-popup to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. goto Appearance ->widgets you will find new sidebar name "Fancy Widget Sidebar", drop your widget there to show it in popup. You can drop any wordpress widget available to your setup. For Example: Archives
4. For Customizing its height, width etc. goto Settings ->Fancy Widget Popup

== Frequently Asked Questions ==

= Will it work on pages? =

It will work on all pages of the site.

= It is customizable? =

It gives option to change its height and width, background color and border color.
